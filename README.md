# ✨ Glow Studio Pro
**Editor de imágenes desarrollado en Python**

> Proyecto de programación con interfaz gráfica, diseñado para editar, mejorar y transformar fotografías de forma sencilla e intuitiva.

---

## 🚀 ¿Qué es este proyecto?
Es una herramienta de edición de imágenes creada totalmente en lenguaje **Python**. Cuenta con funciones similares a programas de edición conocidos (como GIMP), permitiendo aplicar filtros, ajustar colores, recortar y modificar imágenes, mostrando claramente la diferencia entre la imagen original y la editada.

✅ Interfaz gráfica moderna y fácil de usar  
✅ Muestra comparación **Antes / Después** de la edición  
✅ Código abierto y multiplataforma  
✅ Funciona en Windows, Linux y MacOS

---

## 🛠️ Tecnologías utilizadas
- **Python 3.x**: Lenguaje de programación principal
- **CustomTkinter**: Para crear la interfaz gráfica y ventanas
- **Pillow (PIL)**: Librería encargada de procesar y modificar las imágenes
- **GitHub**: Plataforma de alojamiento y control de versiones

> ⚠️ Todas estas herramientas son gratuitas y de código abierto.
